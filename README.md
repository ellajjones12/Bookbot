# Bookbot

A CLI tool that analyzes a text file and generates a reading report.

## Usage

```sh
python3 main.py <path_to_book>
```

## Examples

```sh
python3 main.py books/frankenstein.txt
python3 main.py books/pride_and_prejudice.txt
python3 main.py books/the_whale.txt
python3 main.py books/winnie_the_pooh.txt
python3 main.py books/wuthering_heights.txt
```

## Run All Books at Once

```sh
python3 main.py books/*.txt
```
